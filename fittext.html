FitText Plugin

The FitText plugin is used for making responsive designs easier. Once it is called on an element, it automatically resizes the element's text based on the window's width. The default font size is 1/10th of the window width, but there is also a compresser parameter you can pass in to change this.

The header text above will change dynamically as the window width changes. The slider below can do this instead by changing FitText's compresser value. By default it is set to 1. For this page, however, the default is set to 2. Since this value is compressing the text, a higher value will give smaller text and a smaller value will give larger text.
